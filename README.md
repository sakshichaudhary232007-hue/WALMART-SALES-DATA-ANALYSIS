# WALMART-SALES-DATA-ANALYSIS
# 🛒 Walmart Sales Data Analysis using SQL

## 📌 Project Overview
This project analyzes Walmart sales data using SQL to uncover valuable business insights. The analysis focuses on customer behavior, sales trends, product performance, and revenue generation by writing optimized SQL queries.

The goal of this project is to demonstrate practical SQL skills including data exploration, aggregation, filtering, joins (if applicable), Common Table Expressions (CTEs), window functions, and business problem-solving.
